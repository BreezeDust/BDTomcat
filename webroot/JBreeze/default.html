JBreeze是什么

JBreeze是个JavaScript DHTML工具箱，设计之初用于RIA程序开发，在原有HTML DOM上进行扩展和封装了许多实用功能，兼容浏览器,添加了部分特效和事件管理
并且JBreeze支持AJAX，简化你的前端开发，方便简洁的管理你曾经的项目模板。
  • 如果你是一个初学者，那么JBreeze能帮助你快速构建网页元素，JBreeze拥有基础GUI组件和常用特效，面向对象式编程
  • 如果你是一个经验丰富的开发者，JBreeze能协助你快速封装并且管理你的模板，下次使用只需new一下

几个简单的例子

使用JBreeze只需要引入JBreeze.js和JBreeze.css就可以了

                    <link href="src/Style/JBreeze.css" rel="stylesheet" />
                    <script src="src/JBreeze.js"></script>
1.快速构建一个窗口,并且赋予可拖动功能 
                    var WF = new JBreeze.GUI.Frame();//创建一个窗口
                    WF.EF_baseMove(true);//添加可拖动效果



2.快速构建一个对话框,赋予可拖动效果，并且绑定确定按钮 
                        var msg = new JBreeze.GUI.MSG(null, "测试", "JBreeze MsgBox!!");//创建一个对话框
                        msg.EF_baseMove(true);//添加可拖动效果
                        msg.onConfirm(function () {//绑定确定按钮事件
                            alert("您点击了确定按钮");
                        });



3.利用GUI基本控件，创建一个登陆框 
                    var login = new JBreeze.GUI.Frame();//创建一个窗体
                    login.setSize(400,270);//设置长宽
                    login.JBTitle_.setTitleName("Login");//设置名字

                    var names = new JBreeze.GUI.Text(null, "name:");//创建一个用户名框
                    var passwd = new JBreeze.GUI.Text(null, "password:");//创建一个密码框
                    passwd.JBlabeName_.setWidth(100);//设置字体宽度
                    names.setPositionXY(50, 60);//设置用户名框位置
                    passwd.setPositionXY(10, 80);//设置密码框位置

                    var buLogin = new JBreeze.GUI.Button(null, "登陆");//创建一个登陆按钮
                    buLogin.setSize(100, 35);//设置长宽
                    buLogin.setPositionXY("35%",120);//设置位置
           

                    login.EF_baseMove(true);//可移动
                    login.append(names);//将各元素加入窗体
                    login.append(passwd);
                    login.append(buLogin);



4.利用JBreeze.extendOBJ()函数封装上面构建的模块，然后我们实例化一个对象 
                        JBreeze.extendOBJ({
                            Login: function () {

                           /*----[例3的代码]-----*/

                                return login;//那个登陆框
                            }
                        });

                        var LG=new JBreeze.GUI.Login();//最后我们运行一个实例

对象的创建

JBreeze提供了丰富的对象创建方式，凡是JBreeze对象，都可以使用JBreeze的一切属性和方法
1.使用new的方式,可创建JBreeze对象，JBreeze提供了GUI基础组件
                    var jbobj = new JBreeze.GUI.Frame();//创建一个Frame()



2.使用JBreeze.OBJ()的方式创建JBreeze对象,针对html标签
                     var jbobj = new JBreeze.OBJ(null, "div");//创建一个div的JBreeze对象



3.如果你已经用html构建好一个基础结构,或者使用像JQuery等其他框架创建好对象,
那么您也可以使用JBreeze("ID名")的方式创建JBreeze对象
                    <body>
                        <div id="A" style="width:100px; height:100px; background-color:#969490;"> </div>

                            <script type="text/javascript">
                                var jbobj = JBreeze("A");//注意没有new,
                                jbobj.EF_isShaDow();//因为是JBreeze对象，所以可以使用JBreeze特效,这里我们给于阴影效果
                                jbobj.EF_baseMove(true);//给这个div可拖动的效果
                            </script>
                    </body>

对象的操作

JBreeze为每个JB对象提供了方便操作的方法和属性，列表如下:

方法

方法 描述
setID(idStr)
getID()		 设置和得到ID值 既HTML DOM的id属性
setClassName(className) html dom的class属性,用于CSS选择样式
innerHTML(htmlStr) 和innerHTML对应
setSize(width, height) 设置宽度和高度,可接收的参数:数值:px, 如(100,"100px"),前面的100也是100px
字符串:如"100px","10%","1em"等
setWidth(height)
getWidth()		 设置和得到宽度，参数同上
setHeight(height)
getHidth()		 设置和得到高度，参数形式同上
setPositionXY(Left, Top) 设置位置，如果没有设置position，默认为relative
setPositionX(Left) setPositionY(Top) 设置水平和垂直位置，如果没有设置position，默认为relative


属性

属性 描述
_name 私有属性,JBreeze对象名称
ID 就是HTML节点的id值，一般没有设的话，由JBreeze统一管理，编号从JB1000开始
dom 指向原生的HTML DOM,JBreeze没有提供的功能,你可以使用原生JS编写
JB+单词+_ 如:JBTitle_ JBreeze GUI属性，指引内部子对象,方便开发者快速定位JB内部JB子对象
JB+任意字符+S 如:JBTitleS JBreeze GUI动态属性(为数组)，该属性由append(),removeAll(),removeThis()等函数动态创建和移除
比如:var frame = new JBreeze.GUI.Frame(); frame.JBTitleS[0].JBLabelS[0].SetTextValue();可动态导航到
窗体的标题栏的文本对象,该链式动态属性，目前只有VS2012可动态监测出并智能提示，能提高你的编程速度。
JB+html名_+空或其他字符 如:JBTitleS JBreeze.OBJ()产生的对象名


全局JB对象及其静态方法

对象或方法 描述
JBreeze.JBBody 对body的封装，是个JB对象
JBreeze.onReady() 事件，和JQuery的Ready()功能一样，在onload()事件前执行
JBreeze.ImageLoad(urlArry) 预加载图像.....
JBreeze.trim(str) 去除首尾空格
JBreeze.styleToCss(StyleStr) 将驼峰式转换成CSS样式 如"backgroundColor"--->"background-color"
JBreeze.cssToStyle(cssStr) 将CSS样式写法转换成驼峰式 如"background-color"-->"backgroundColor"
JBreeze.getCSSValue(obj, nameStr) 得到CSS属性值，检查顺序为，stly，如果没有找到，再查找样式表
obj为HTML dom,这里不是JB对象
JBreeze.getHttpRequest() 得到一个兼容性的XMLHttpRequest
CSS操作函数

1.setCss(cssObj)
JBreeze setCss()函数可接收各种参数
参数可为"width:200px; height:100%;" 字符串格式，也可以为{"width":"200px", "height":"100%;" }
或者为{width:"200px", height:"100%;" }，但要注意{"background-color":"#000"}等中间有-符号的要加引号
当setCss("");表示清除style的所有内容 
                    var jbobj = new JB.GUI.Frame();
                    jbobj.setCss("width:200px; height:100%;");//正确格式  但要清除以前设置的style，不推荐使用
                    jbobj.setCss({ "width": "200px", "height": "100%;", "background-color": "#000" });//正确格式
                    jbobj.setCss({ width: "200px", height: "100%;" });//正确格式


1.setCss(cssName)
参数为空时返回style属性的值,有参数时，请按如下格式:"background-color"和"backgroundColor"都可以
                    jbobj.getCss();
                    jbobj.getCss("background-color");
                    jbobj.getCss("backgroundColor");

JBreeze事件管理

JBreeze模仿C#的事件委托机制，可动态的添加事件，和删除事件，并且维持一个事件列表

1.绑定事件
                    var JBF = new JBreeze.GUI.Frame();
                    JBF.onclick(function () {
                        alert("ok");
                    });
                    //或者
                    var afun = function () { alert("ok") };
                    JBF.onclick(afun);



2.事件流
                    var JBF = new JBreeze.GUI.Frame();
                    var a = function () { alert("a") };
                    var b = function () { alert("b") };
                    var c = function () { alert("c") };
                    JBF.onclick(a);
                    JBF.onclick(b);
                    JBF.onclick(c);



3.删除事件
                    var JBF = new JBreeze.GUI.Frame();
                    var a = function () { alert("a") };
                    var b = function () { alert("b") };
                    var c = function () { alert("c") };
                    JBF.onclick(a);
                    JBF.onclick(b);
                    JBF.onclick(c);
                    JBF.onclick(null, b);



事件列表

事件 描述
JBreeze.onReady(funOption, clearEvent) 类似JQ的Ready事件
funOption:响应函数指针

clearEvent:响应函数指针 时，清除这响应个函数

ondblclick(funOption, clearEvent) 参数同上
onmousedown(funOption, clearEvent) 参数同上
onmousemove(funOption, clearEvent) 参数同上
onmouseup(funOption, clearEvent) 参数同上
onmouseout(funOption, clearEvent) 参数同上
onmouseover(funOption, clearEvent) 参数同上
onkeydown(funOption, clearEvent) 参数同上
onkeypress(funOption, clearEvent) 参数同上
onkeyup(funOption, clearEvent) 参数同上
onload(funOption, clearEvent) 参数同上
基本特效
JBreeze的特效列表如下:

JBreeze特效

特效 描述
EF_isShaDow(color, radiu) 阴影效果
color:阴影颜色
radiu:阴影半径
EF_baseMove(isCanMove, fatherDeep) 拖动效果
isCanMove:true为可拖动，false为不可拖动
fatherDeep:true时为父级对象可拖动，默认为false
EF_XCenter() 水平居中
必须设置长宽，次效果才有效
EF_XYCenter(isFixed) 水平垂直居中
isFixed true 为屏幕中间不随页面滚动 有默认值
必须设置长宽，次效果才有效
EF_loading(html) 图像等加载效果
仅用于<body>, <frame>, <frameset>, <iframe>, <img>, <link>, <script>
html 为你自定义的加载模块
XML和JSON

1.Get(url, isXML, callback)
用于get方式的AJAX请求
+url:请求地址
+isXML:true为XML类型
+callback:成功后执行的函数 形式为function(data):data为请求后的数据，JSON为JSON对象，XML为responseXML 

                    JBreeze.Get("test.php",false, function (data) {
                        alert(data.name);//data为JSON对象
                    });

2.Post(url, isXML, data, callback)
用于POST方式的AJAX请求
+url:请求地址
+isXML:true为XML类型
+data:data 参数对象 {"名字":"值","名字":"值".....}
+callback:成功后执行的函数 形式为function(data):data为请求后的数据，JSON为JSON对象，XML为responseXML 
                    JBreeze.Post("test.php",false,{a:1,b:"test"}, function (data) {
                        alert(data.name);//data为JSON对象
                    });
远程文件加载

Load(url, callback)
您可以使用此函数加载远程或者服务端的文本文件
+url:请求地址,只能为GET方式请求
+callback:成功后执行的函数 形式为function(data):data为请求后的文本数据 
                    JBreeze.Load("file.txt",function (data) {
                        test.innerHTML(data);//将数据放入HTML DOM
                    });
